# Introduction au services cloud AWS

## Objectifs pédagogiques:

À la fin du cours, les étudiants seront capables de :

* Comprendre les concepts fondamentaux du cloud computing
* Déployer des architectures simples sur AWS
* Configurer des réseaux, stockage, calcul et bases de données
* Utiliser certains services data, analytics et IA
* Construire une architecture cloud complète

Chaque section comprend :

* Concepts théoriques
* Présentation des services AWS
* Travaux pratiques (Labs)
